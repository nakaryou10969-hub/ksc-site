# KSC-SITE 開発ログ

## プロジェクト概要
- Next.js 16.2.4 (Turbopack) + Tailwind CSS v4
- 静的エクスポート (`output: "export"`)
- GitHub: https://github.com/nakaryou10969-hub/ksc-site.git
- microCMS でイベント記事を管理

## 技術スタック
- フレームワーク: Next.js (App Router)
- スタイリング: Tailwind CSS v4 (`@import "tailwindcss"` + `@theme`)
- フォント: Noto Sans JP (400/700/900)
- 画像: next/image (unoptimized)
- フォーム: Formspree

## ブレークポイント設定 (`globals.css @theme`)
- `--breakpoint-md: 530px` — 全般的なPC/スマホ切り替え
- `--breakpoint-lg: 850px` — 各メンバーカード（挑戦者・応援団・盛り上げ隊・助っ人団）のPC/スマホ切り替え
- ヘッダーは `min-[1100px]:` で独自に切り替え

## メンバーカード構造

### PC版 (850px以上)
- 挑戦者: 青枠(#001597) + 左に画像 + 右にテキスト
- 応援団: 金枠(#C79500) + 左にテキスト + 右に画像
- 盛り上げ隊: 金枠 + 左に画像 + 右にテキスト
- 助っ人団: 金枠 + 左にテキスト + 右に画像
- 画像: -mobile.png を使用、`absolute inset-0 object-contain object-center`
- minHeight: `clamp(120px, 15vw, 220px)`
- overflow: visible（挑戦者）/ visible（他3つ）

### スマホ版 (850px未満) — MemberCardMobile コンポーネント
- 幅: 90% mx-auto（親divで制御）
- 画像は枠内に収める（overflow: hidden）
- 画像高さ: `clamp(180px, 45vw, 280px)`
- V字のくびれ: clip-pathの三角 + 詳しく見るボタン
- 展開アニメーション: max-height 0→600px, 0.5s ease-in-out
- bgColor: #E3E0DA（セクション背景と同色）
- isolation: isolate（カード間のz-index干渉防止）

### カード内テキスト改行ルール
改行を入れる箇所のみ:
- 挑戦者: 「〜チャレンジャー。」の後
- 盛り上げ隊: 「〜させる方たち。」の後 / 「〜紹介したい！」」の後 / 「〜広げていきたい！」」の後
- 応援団・助っ人団: 改行なし（自然折り返し）

## 注意事項
- PowerShellの `Set-Content` は日本語ファイルを文字化けさせるので使用禁止
- ファイル編集は必ず `str_replace` ツールまたは `fs_write` を使う
- `git checkout <hash> -- <file>` で安全にファイルを復元可能
- push.ps1 は古いコミットメッセージが入っているので、都度 git コマンドで直接プッシュ
