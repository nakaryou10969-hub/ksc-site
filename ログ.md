# ksc-site 作業ログ

## 概要
KANDA Startup Commons（KSC）の公式Webサイト構築プロジェクト。
novolba-siteと同じ技術スタックを使い、独立したサービスとして構築。

---

## 技術スタック
- **フロントエンド**: Next.js 16 + TypeScript + Tailwind CSS v4
- **ホスティング**: AWS S3 + CloudFront（S3ウェブサイトホスティング + CloudFront）
- **CMS**: microCMS（別アカウント、サービスID: `ksc`）
- **お問い合わせ**: Formspree（別アカウント）
- **CI/CD**: GitHub Actions
- **リポジトリ**: https://github.com/nakaryou10969-hub/ksc-site

## トラブルシューティング記録

### CloudFrontで記事ページが404になる問題（長期）
**原因**: `S3OriginConfig` に `OriginReadTimeout: 30` という不正なフィールドが混入し、CloudFrontがレガシーモードで動作。キャッシュが正しく無効化されない状態が続いた。

**解決策**: CloudFrontディストリビューションを作り直し（`ksc-currentsite`）、S3ウェブサイトエンドポイントをオリジンとして使用。

**教訓**:
- CloudFrontの設定変更はAWS CLIで行うと不正フィールドが混入しやすい
- 問題が解決しない場合はディストリビューションを作り直すのが最も確実
- ローカルからS3に手動でファイルをアップロードするとビルドハッシュが混在して壊れる
- **今後は必ずGitHub Actions経由でデプロイ**（`git push` するだけ）
