# ksc-site 作業ログ

## 概要
KANDA Startup Commons（KSC）の公式Webサイト構築プロジェクト。
novolba-siteと同じ技術スタックを使い、独立したサービスとして構築。

---

## 技術スタック
- **フロントエンド**: Next.js 16 + TypeScript + Tailwind CSS v4
- **ホスティング**: AWS S3 + CloudFront
- **CMS**: microCMS（別アカウント）
- **お問い合わせ**: Formspree（別アカウント）
- **CI/CD**: GitHub Actions
- **リポジトリ**: https://github.com/nakaryou10969-hub/ksc-site
